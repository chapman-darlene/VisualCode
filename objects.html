JavaScript Objects

\"JavaScript Objects fundamental datatype is the object.\" It allows yo to store and retrieve values by its name. It is a collection of unordered properties. The property names are strings, so it makes objects able to map strings to values. This feature is similar to another structure called an associative array.
JavaScript object also inherit the properties of another object, known as a prototype. This \"prototypal inheritance\" is a key feature of JavaScript.

Each JavaScript object has a name and value, as well as property attributes. Every object has three associated object attributes. Here are basic object examples:

  • Here is one way to an empty object:
  • var object1 = new Object();
  • You can also use a method to create a new object, and pass values as parameters into an object
  • var object2 = Object.create({x:1, y:2});
  • An object array is created like this
  • var array1 = new Array();
  • This is a date object
  • var date1 = Date();

You can use array notation to access properties of an object with string expressions.
