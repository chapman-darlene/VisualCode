CSS Transition

CSS transition are a part of the CSS3 language to add dynamic content to a web page.

CSS Transition Property

CSS transitions create a way to change a property value over a certain duration of time. This gradual change is usually more aesthetically pleasing to the viewer. Here is an example of a value change to a box with and without the transition applied

Box change without Transition

Box change with Transition

Transitions have four components or features. They are:

  • Property: This determines what is to be transitioned, transformed or animated.
  • Duration: This is the length of time between the beginning and end of the transition.
  • Duration-Timing-function: This determines the speed of the transition
  • Delay: This is an optional feature that determines the start time of the transition.

Here is an example of what the code would look like in CSS:

  • .box{
  • transtion: all 3s ease-in-out;
  • }

CSS Duration-Timing-function

This feature may be considered to have the greatest impact upon the transition. Hover over the circles to see how they change:

Linear: Creates an effect with the same speed through out the transition

Ease: Creates an effect that starts slow, increases in speed, then ends slowly

Ease-in: Creates an effect that starts slowly

Ease-out: Creates an effect with a slow ending

Ease-in-out: Is an effect that starts and end slowly

Cubic-bezier: The user can define their own values
