Variables

To understand JavaScript, we need to understand how JavaScript stores information. The simplest way is through creating variables. A variable in JavaScript is easily created by placing the keyword \"var\" in front of a unique name, known as an identifier. For example: var text = \"This is some text\". Variables must follow simple rules, such as:

  • Variables may include letters (a-z, A-Z) or numbers (0-9). It only allows the symbols $ (dollar sign) and _ (underscore)
  • No other characters, spaces or punctuation is allowed.
  • The first character can only be a letter, the $ sign or _
  • Names are case-sensitive. For instance: Text, text and TEXT are not the same identifiers.
  • There are no set limits on variable names, but they should reflect the item to be defined.
  • Camel-case variable names are strongly suggested in defining a variable name, such as: var = firstName.

Functions & Parameters

The primary purpose of JavaScript is to provide dynamic content to a website. This is made possible primarily through the use of functions. JavaScript is a C-based program, high-level programming language, so it is relatively easy to learn in comparison to many lower-level languages. Understanding how to create simple to complex functions is an essential skill for any JavaScript programmer.
Functions are used to separate and organize code into smaller, reusable blocks. A simple function is created as follows:

  • <script>
  • function sample(a, b) {
  • return a*b
  • }
  • </script>

This function passes two parameters (a and b). It then acts upon the variables by multiplying the variables and returning the result back to the main flow of the program. Parameters make it possible for the function to return different values when the function is once again called.

Arrays

Arrays are designed to save a multitude of values in a single variable. Each value is delimited and indexed into a point of memory that is accessible by the variable name and index number. For example:

  • Creating an array name is easily accomplished using the following syntax:
  • arrayName = new Array();

  • The shorthand form is:
  • arrayName = [];
  • arrayName = ['Element 1', 'Element2'];

  • JavaScript uses the push method to assign element values, for instance:
  • arrayName.push("Element 1");
  • arrayName.push("Element 2");

  • Another option is to place the element into a specific location, such as:
  • arrayName[0] = "Element 1";
  • arrayName[1] = "Element 2";

  • To locate an element in an array, one can use the indexOf or the forEach method.
  • indexOf Method:
  • animals = ['cat', 'dog', 'cow', 'horse'];
  • offset = animals.indexOf('cow');

  • forEach Method:
  • animals.forEach(output);
  • function output(element, index, array){
  • document.write("Element at index " + index + " has the value " + element + "<br>);}

Because JavaScript is a loosely scripted language, an array can hold a different variable types, such as strings, numbers, characters, etc.

Associative Arrays

When working with large multi-dimensional arrays, it is often useful to transform the array into an associative array, also known as objects as associative arrays. Associate arrays allow elements in an array to be referenced by name rather than by an integer. In JavaScript, that option is achieved by creating an object with properties that act as an associative array.
To create this array, a block of elements is defined within curly braces. Each element is placed with a key on the left and the contents on the right, separated by a semi-colon, as seen below.

  • <script>
  • sports = { "golf": "Needs a club, golf ball, and 18-holes",
  • "tennis": "Needs a racket, tennis ball, court and net",
  • "basketball": "Needs a hoop, basketball, and court"
  • "football" : "Need helmet and pads, football, and field"}

  • //Use a for loop to access the information in the array
  • for(sport in sports)
  • document.write(sport + " = " + sports[sport] + "<br>)
  • </script>
